# Ejecutar en desarrollo

1. Clonar el respositorio
2. Ejecutar

```
yarn install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```

5. Iniciar seed
```
llamar al servicio http://localhost:300/v1/seed
```

## Stack usado
*Mongo DB
*Nest
